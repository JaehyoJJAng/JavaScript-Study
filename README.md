## Node.js 설치하기
Node.js 서버를 설치하기 위하여 <a href="https://nodejs.org/en/">다운로드</a> 버튼을 눌러주세요

> <strong>Node.js 설치 위치 확인하기</strong><br>

```
$ which node
/usr/local/bin/node
```
<br>

> <strong>Node.js 실행하기</strong>

```
$ echo -e "console.log("Hello");\nconsole.log("Bye") >| ./index.js
$ node ./index.js
Hello
Bye
```
